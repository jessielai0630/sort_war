選擇排序法，最基礎的排序法之一。以下為執行步驟:
1. 從還未排序好的數字中找到其中的最小值(最大值)
2. 若為由小到大(由大到小)排序則將最小值(最大值)放到最左邊並標示為已排序
3. 重複步驟 1~2 後即可完成排序

void selectionSort(int a[]){
 for(int i=0;i<a.length();i++){
   max=i;
   for(int j=i+1;j<a.length();j++){
     if(a[j] > a[max]){
      max=j;
     }
   }
   if(max!=i){
    swap(a[max], a[i]);
   }
 }
}
