BUBBLE SORT

泡沫排序法，又稱為交換排序法，是最常見也最容易實作的演算法。以下為執行步驟(陣列長度為n):
1. 從第一個元素開始逐一比較相鄰的兩個元素
2. 若兩個元素的大小順序有誤則進行交換，若順序正確則保持不動
3. 結束時可以確定最後一個位置的資料是正確的
4. 重複步驟 1~3 n-1次後即可完成排序

  for(int j=0 ; j<(4-i-1) ; j++){
   if(a[j] < a[j+1]){
    swap(a[j], a[j+1]);
   }
  }
 }
