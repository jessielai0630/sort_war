插入排序法，原理是將未排序的資料逐一與已排序的資料做比較，找到對的位置並插入到已排序的資料中。

void insertionSort(int a[]){
 for(int i=1;i<a.length();i++){
   target=i;
   for(int j=i-1;j>=0;j--){
     if(a[target] > a[j]){
      swap(a[target],a[j]);
      target=j;
     }
   }
 }
}
